Daydream Air Mouse

Turn a Google Daydream controller into a wireless USB presentation mouse using an ESP32-S3.

  1. 1 Connect your ESP32-S3 board to your computer via USB.
  2. 2 Select your board below, then click Install Firmware.
  3. 3 Select the serial port, wait for the flash, then pair your Daydream controller!
⚠️ Web Serial is not supported in this browser.
Please use Google Chrome or Microsoft Edge on desktop.
Tip: If the board isn't detected, try holding the BOOT button while plugging in the USB cable.
